# LFEM_TEST

A lightweight tensor-mesh solver for partial differential equations (work in progress).

This repository contains a simple tensor-mesh based high-order FD/FV/DG framework for hyperbolic equations. The project is mainly for personal research and experimentation.

## Warning

This repository contains personal research code. Some parts of the implementation are still under development and testing, and certain algorithms may be experimental or not fully validated.

If you notice any issues or have questions about the implementation, you are welcome to contact me or start a discussion.

Email: **[lightpl@stu.xmu.edu.cn](mailto:lightpl@stu.xmu.edu.cn)**

## Dependencies

The following libraries are required:

* Eigen: https://gitlab.com/libeigen/eigen/-/releases/3.4.1
* OpenMP: https://www.openmp.org
* Intel MKL: https://www.intel.cn/content/www/cn/zh/developer/articles/guide/intel-math-kernel-library-intel-mkl-2019-install-guide.html

## Compilation

The code has been tested on the computer **c0196** at Xiamen University.

You can compile the project using the provided `Makefile`. Before compiling, please modify the paths of the following variables in the Makefile:

* `Eigen_DIR`
* `MPI_Lib_DIR`
* `MPI_Include_DIR`
* `MKL_Lib_DIR`
* `MKL_Include_DIR`

### Build

Run

```bash
make
```

This command will:

* Compile all source files except those in `src/unsupported`
* Generate executable files for:

  * `main.cpp`
  * all `.cpp` files in `src/example1D`
  * all `.cpp` files in `src/example2D`

### Clean

Run

```bash
make clc
```

This command deletes all files generated by the `make` command.
